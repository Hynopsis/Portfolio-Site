Last One Out Android Game

A simple 3D driving survival game made for Android. Your goal is to rescue survivors from their houses and return them to designed drop-off points on the edges of the map. Each level has 5 stages, each with a larger number of cities and survivors to collect. Naturally this increases the difficulty. Each level and it's stages are generated using a seeded random number generator, so you can your friends can pick the same level number and play the same random stage. The Youtube link below shows me playing the first stage. It is still a work in progress, but will eventually be released on the Play Store.

  • Resources: Java, Libgdx, Bullet, Blender, Gimp
  • Date: 5/2016
  • Links: Youtube Github

Stranded Visual Basic Game

A small roguelike game I made using VB.Net. It features a random 2d world of mountains and trees that stand in the way of your lost crewmates. You have to manage your stamina, which decreases a little for each move, more for each tree cut down, and a lot for mining. You regain stamina by hunting pigs and eating meat. Survive bear attacks and find your missing crew.

  • Resources: VB.Net, Gimp
  • Date: 5/2015
